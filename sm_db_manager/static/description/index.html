Module Video Overview

Features

1 Quick Backup with Progress Bar

One-click database backup (ZIP) directly from the user dropdown menu. A beautiful animated progress overlay shows real-time status: Dumping database..., Compressing files..., Building archive... — completing with a confirmation when the download starts. Uses cookie-based completion detection for accurate progress tracking.

2 DB Info Panel

See your database name, size, and last backup time at a glance — right in the user dropdown menu. The info panel refreshes every time you open the menu, showing always up-to-date information. Includes a note about ZIP compression ratio.

3 Copy DB Name

Copy your database name to the clipboard with a single click. Perfect for quickly pasting into terminal commands, backup scripts, or support tickets. Shows a success notification with the copied database name.

4 Database Manager Link

Quick access to Odoo’s built-in Database Manager page. Opens in a new tab for full backup, restore, duplicate, and delete operations without leaving your current page.

5 Backup History Logging

Every backup is automatically logged with filename, format (ZIP/dump), user who triggered it, database size, and timestamp. Full audit trail so you always know when & who backed up the database.

Role-Based Access

Two access levels: User (view info, copy name) and Manager (backup, DB Manager link). Configurable per user in Settings.

Animated Progress

Beautiful progress overlay with pulsing database icon, gradient progress bar, and dynamic status text during backup.

Admin Only

3-layer security: frontend group check, backend group verification, and model-level ACL. Non-authorized users see nothing.

Smart Compression

ZIP backup compresses your database significantly. A 146 MB raw database typically becomes ~26 MB. Compression note shown in the info panel.

Live Refresh

DB info panel refreshes every time you open the user menu. Always see the latest database size and last backup timestamp.

Zero Dependencies

Only depends on the web module. No external libraries, no extra configuration. Install and use immediately.
